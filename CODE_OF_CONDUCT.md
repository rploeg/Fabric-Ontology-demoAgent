# Code of Conduct

We are committed to fostering a welcoming and respectful environment. Please:

- Be friendly, patient, and considerate.
- Assume good intent and seek clarification before escalating disagreements.
- Provide constructive feedback; focus on ideas, not people.
- Avoid harassment or exclusionary behavior.

If you observe unacceptable behavior, report it to the maintainers so we can address it promptly. Participation implies acceptance of this Code of Conduct.
